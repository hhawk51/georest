The FDO Schema representation allows clients to obtain metadata for the resource.

## FdoSchema Renderer ##

To enable the Fdo Schema representation of a resource, you need to add a new representation to the resource's restcfg.xml configuration file, with its renderer set to "FdoSchema".  The recommended URL pattern for this representation is ".schema".

Example:

```
<Resource uripart="property">
   <Source type="MapGuide">
    <FeatureSource>Library://Samples/Sheboygan/Data/Parcels.FeatureSource</FeatureSource>
    <FeatureClass>SHP_Schema:Parcels</FeatureClass>
   </Source>
   <Representation renderer="FdoSchema" pattern=".schema" />
</Resource>
```

To get the property resource's FDO Schema:

`http://localhost:99/rest/data/property/.schema`
